# Flask CRUD Application using MySQL

A simple CRUD (Create, Read, Update, Delete) web application built using Flask (Python) and MySQL.  
This project demonstrates backend development, database connectivity, and template rendering using Flask.

---

## Features
- Create new records
- View all records
- Update existing records
- Delete records
- Flask backend with MySQL integration

---

## Tech Stack
- Python
- Flask
- MySQL
- HTML (Jinja Templates)

---

flask_crud/

pip install flask mysql-connector-python
